About Us

Progate provides online coding learning services. Progate is here with the mission of 'Coding for All' and wants to reach both IT enthusiasts and the general public, so that more users will have the skills that are most in demand in the industrial era 4.0.

Especially for the Indonesian version of Progate, currently there are packages to learn coding in 5 programming languages (HTML & CSS, Javascript, SQL, React, and NodeJS) and 26 lessons (lessons). Globally, Progate provides 77 coding lessons in 15 programming languages.
